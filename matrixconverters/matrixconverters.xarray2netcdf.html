matrixconverters.xarray2netcdf module¶

matrixconverters.xarray2netcdf.xr2netcdf(dataset, filepath, mode='w', engine='netcdf4', compressed=True, complevel=2)¶

save dataset as netcdf-file to filepath using the given compression level

Parameters

  • dataset (Dataset) – the xarray-dataset to store

  • filepath (str) – the path where the netcdf-file shold be stored

  • mode (str) – write (w) or append (a)

  • engine (str) – the engine to use (default is netcdf4)

  • compressed (bool) – if False, the data-variables are not compressed

  • complevel (int) – the compression-level between 1 and 9 1 is faster, 9 uses a more efficient compression, but is much slower 2 is in general a good compromise
