cythoninstallhelpers.get_version.get_version(package_name, setupfilepath, package_dir='src', versionfile='_version.py')¶

Parse the version number from a version file

Parameters

  • package_name (str) – the name of the package the version is searched for

  • setupfilepath (str) – the filepath, where the setupfile can be found

  • package_dir (str) – the subdirectory with the source code of the version file

  • versionfile (str) – the filename of the python module with the version number

Return type

str
